## Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn
- Google Gemini API key (optional, for AI features)

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd fitness-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   ```
   
   Edit `.env` and add your Google Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your-api-key-here
   VITE_ENABLE_AI=true
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173`

### Getting a Google Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Create a new API key
4. Copy the key to your `.env` file

**Note**: The app works without an API key using predefined workout templates, but AI features will be disabled.

## Customization

### Adding New Workout Types
1. Update the `types.ts` file with new workout types
2. Add corresponding colors in CSS files
3. Update the AI service to generate appropriate exercises

### Modifying the Questionnaire
1. Edit the `questions` array in `Questionnaire.tsx`
2. Update the AI prompt in `aiService.ts` to use new data
3. Adjust the UI components as needed

### Styling
- Global styles and color variables are in `src/index.css`
- Component-specific styles are in individual `.css` files
- Uses CSS custom properties for consistent theming
